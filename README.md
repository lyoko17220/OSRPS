# OSRPS


Docker OSRPS

Pré-requis:
 - Python3
 - Pip
 - Pyshark
 - Tcpdump
 - Scapy
 - Dig
 - Nslookup
 - Nmap
 - Awk

## Usage
*Via PyCharm*

- Ajouter un interpreteur
- Docker > lyoko17220/osrps-standalone
- Changer l'éxécutable `python` en `python3`
